Exportar / Importar Datos

Cada tipo de dato tiene su propio archivo JSON. Descarga, edita y vuelve a importar cualquiera de ellos para actualizar tu tienda.

📦 Productos

products.json — todos los productos de la tienda

🗂 Categorías

categories.json — catálogos / colecciones

🧾 Pedidos

orders.json — historial de todas las órdenes

💳 Métodos de Pago

payment_methods.json — métodos habilitados e instrucciones

⚙️ Configuración

settings.json — tema, fuente, textos, logos y más. Importa otro JSON con el mismo formato para cambiar el diseño.
